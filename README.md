# Описание проекта
Командная работа по верстке.

Проектом является статичный сайт. Сайт является адаптивным, умеет подстраиваться под экраны разных разрешений.

### **Используемые технологии:**
* Адаптивная верстка;
* БЭМ-методология;
* Flexbox;
* Grid Layout.
* Hover-эффекты;
